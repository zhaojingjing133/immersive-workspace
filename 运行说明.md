# 如何运行项目

## 第一步：打开终端/命令行

在Mac上：
- 按 `Command + 空格键` 打开Spotlight
- 输入 `Terminal` 或 `终端`
- 按回车打开终端

在Windows上：
- 按 `Win + R`
- 输入 `cmd`
- 按回车打开命令提示符

## 第二步：进入项目目录

在终端中输入以下命令（根据你的实际路径调整）：

```bash
cd /Users/jingjing/1
```

或者如果你已经在项目文件夹中，可以跳过这一步。

## 第三步：安装依赖包

```bash
npm install
```

这一步会安装所有需要的依赖包（React、MUI、PDF库等），可能需要几分钟时间。

## 第四步：启动开发服务器

```bash
npm run dev
```

启动后你会看到类似这样的信息：
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
```

## 第五步：在浏览器中打开

开发服务器启动后，**浏览器会自动打开**，或者你可以：

1. **自动打开**：如果设置了 `open: true`，浏览器会自动打开
2. **手动打开**：在浏览器地址栏输入：
   ```
   http://localhost:3000
   ```
   或者
   ```
   http://127.0.0.1:3000
   ```

## 完整命令流程（复制粘贴）

```bash
# 1. 进入项目目录
cd /Users/jingjing/1

# 2. 安装依赖（只需要第一次运行）
npm install

# 3. 启动开发服务器
npm run dev
```

## 常见问题

### 如果提示 "npm: command not found"
说明没有安装 Node.js，需要先安装：
- 访问 https://nodejs.org/
- 下载并安装 LTS 版本

### 如果端口 3000 被占用
服务器会提示使用其他端口（如 3001），在浏览器中打开新提示的地址即可。

### 如果想停止服务器
在终端中按 `Ctrl + C`（Mac 上也是 `Ctrl + C`）
